1. Ответ на первое задание находится в файле first_task.txt
2. Ответ на второе задание находится в файле second_task.txt

3. Для полноценной работы программы cbr_currency.py для 3-ого задания требуется предварительно поставить несколько фреймворков командами:
python -m pip install requests
python -m pip install lxml  (требуется для парсинга xml-файла)
python -m pip install bs4   

Для запуска программы требуется через терминал перейти в директорию с файлом cbr_currency.py (cd "directory") и запустить файл командой:
python -m cbr_currency

В поле вывода программы позволил себе последней строкой дописать время выполнения

Для изменения интервала времени(не 90 дней, а,например, 120) требуется открыть файл cbr_currency.py любым редактором и изменить значение в скобочках (days=90) на необходимое. Находится на 60-й строке

Валюты округляются до 3 знаков после запятой.
